  • How to embed a QRCode Link on your site.

    Place this code on the page you wish to render a QRCode: 

    <img src="http://qrcodelinkulator.appspot.com/128" />

    You can specify image size like this: 
    <img src="http://qrcodelinkulator.appspot.com/128" width="128" height="128" />

    NOTE: the number in bold represents both width and height since QR Codes are square.

  • How to embed a specific link image on your site

    Place this code on the page you wish to render a QRCode: 

    <img src="http://qrcodelinkulator.appspot.com/128?http://google.com" />
